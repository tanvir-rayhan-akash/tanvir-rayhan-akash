## About

I build low-level Android tooling — kernel-side modules, root-environment utilities, and network proxy systems. My current focus is **Proxy4KSU-TRA**, a KernelSU module that integrates the Xray proxy core directly into the Android kernel-su environment.

I work primarily in shell scripting and Android system internals, with particular interest in process synchronization and network reliability at the OS level.

### [Proxy4KSU-TRA](https://github.com/tanvir-rayhan-akash/Proxy4KSU-TRA)

A KernelSU module that embeds the **Xray** proxy core into the root environment, providing system-level network proxying without a standalone app layer.

**Engineering highlights:**
- Resolved race conditions between two concurrent network-state monitors using a shared lock mechanism
- Added debounce logic to filter transient network blips and prevent false disconnect triggers
- Tuned buffer sizing in the base config for stability under load
