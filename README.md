### [`Proxy4KSU-TRA`](https://github.com/tanvir-rayhan-akash/Proxy4KSU-TRA) — KernelSU × Xray

> *"when your ISP tries to block you but you build a kernel module" 💀*

A **KernelSU module** that embeds the Xray proxy core directly into the root environment — no app layer, no fluff, just raw OS-level networking.

| what i fixed | how i fixed it |
|---|---|
| 🐛 race conditions between network monitors | shared lock mechanism + mutex logic |
| 📡 fake disconnect triggers on signal blips | debounce logic + threshold filtering |
| 🔧 buffer overflow under heavy load | tuned tun2socks + xray config sizing |
| 💀 startup timing hell after ROM flash | adaptive polling + backoff system |

`Shell` `KernelSU` `Xray-core` `tun2socks` `Android 16` `SuSFS`
